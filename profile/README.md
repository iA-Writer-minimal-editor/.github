# iA Writer — Portable Minimalist Editor with Focus Mode

## About iA Writer

iA Writer provides a complete distraction‑free writing environment for users who want to focus on words, not toolbars. iA Writer combines focus mode (highlighting current sentence), syntax highlight, and a clean interface. iA Writer supports Markdown shortcuts, customizable typewriter sound, word count, and document library. iA Writer enables users to write without distraction and export to Markdown, PDF, Word, HTML. iA Writer runs on desktop platforms. iA Writer is trusted by authors, journalists, bloggers, students, and anyone who writes.

## Deep Dive — Minimalist Editor Notes

Focus mode: dim everything except current sentence, fade previous lines
Syntax highlight: nouns, verbs, adverbs, adjectives, prepositions
Markdown shortcuts: inline formatting without breaking flow
Typewriter sound: optional mechanical keyboard audio feedback
Document library: local folders, iCloud Drive, Dropbox, external sync
